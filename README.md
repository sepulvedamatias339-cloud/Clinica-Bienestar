# Clínica Bienestar

Sistema web desarrollado con Node.js, Express y EJS para la gestión básica de pacientes en una clínica.

## Funcionalidades

- Inicio de sesión
- Dashboard principal
- Registro de pacientes
- Visualización de pacientes
- Eliminación de pacientes
- Cierre de sesión
- Interfaz web responsive básica

---

## Tecnologías utilizadas

- Node.js
- Express
- EJS
- HTML5
- CSS3
- GitHub
- Render

## Instalación local

1. Clonar repositorio

```bash
git clone https://github.com/sepulvedamatias339-cloud/Clinica-Bienestar.git
```

2. Instalar dependencias

```bash
npm install
```

3. Ejecutar servidor

```bash
node app.js
```

4. Abrir navegador

```bash
http://localhost:3000
```

---

## Objetivo del proyecto

El objetivo del sistema es permitir la gestión básica de pacientes mediante una aplicación web simple, aplicando conceptos de desarrollo backend y frontend utilizando Node.js y Express.
